# Azure Chatbot 🤖

Simple chatbot built with Node.js and Express, deployed on Azure App Service.

## Features
- Chatbot with basic responses
- REST API (`/chat`)
- Simple frontend UI
- Azure cloud deployment

## Tech Stack
- Node.js
- Express.js
- HTML/CSS/JS
- Azure App Service
